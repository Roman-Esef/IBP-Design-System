Иллюстрации

Библиотека продуктовых иллюстраций: тайловые (для навигационных плиток NavTile и дашбордов), иллюстрации состояний (ошибки, пустые экраны) и фоновая декоративная иллюстрация. Как на странице Иконки — клик по карточке копирует имя.

Слот иллюстрации

Иллюстрация вставляется через слот .illu с именем в data-illu. Скрипт ds-illustrations.js подставляет реальный SVG из assets/illustrations/ в тот же слот.

96×96 — дефолт (NavTile)

140×140 — пустые состояния

64×64 — компакт

имени нет в библиотеке — заглушка

Правила

  • слот квадратный, SVG внутри масштабируется через object-fit: contain — не обрезается; исходные пропорции тайловых файлов 195×140, у крупных (состояния) — почти квадратные;
  • иллюстрации декоративны — слот всегда с aria-hidden="true";
  • имена — латиница в kebab-case, см. полный список ниже; неизвестное имя рисует штриховую заглушку с этим именем;
  • цвета внутри SVG — фиксированная палитра библиотеки, перекраска через currentColor не применяется (в отличие от иконок).

Тайловые иллюстрации

195×140, для NavTile и карточек дашборда. 32 штук.

Иллюстрации состояний

Крупные иллюстрации для полноэкранных ошибок и недоступности сервиса.

Фоновая иллюстрация

Декоративный фон 1066×777 — используется как background-image на весь блок (не через слот .illu, у него другие пропорции и назначение).

Пример: background-illustration.svg, растягивается через background-size: cover.
