DatePicker

Календарь — всплывающая поверхность выбора даты или диапазона дат. Поднимается полями InputDate и InputDateRange при фокусе или по кнопке-календарю. Material-подход, собранный на токенах и шрифтах ДС. Неделя начинается с понедельника, локаль — русская.

Конструктор

Соберите календарь: режим выбора и футер. Выбор живой — кликайте по дням, листайте месяцы стрелками, откройте выбор года кликом по заголовку.

Использование

DatePicker — единственный способ выбора даты указателем. Он не вводится в поток формы сам по себе, а всплывает над полем InputDate / InputDateRange (floating) либо встраивается в панель или модалку (inline). С клавиатуры дату можно ввести и без календаря — прямо в поле по маске.

Одиночная дата — выбор одного дня для поля InputDate.
Диапазон — начало и конец периода для InputDateRange, промежуток подсвечен.
Inline — встроенный в модалку/панель, с кнопками подтверждения.

Когда использовать

  • выбор конкретной даты указателем: подписание, транш, плановая дата;
  • выбор периода «с … по …» — режим диапазона для InputDateRange;
  • выбор месяца целиком (отчётный период) — режим «Месяц».

Для ручного ввода одной даты по маске поле InputDate достаточно и без открытия календаря. Выбор из справочника значений — InputAutocomplete, не календарь.

Когда не использовать

  • для быстрого ввода известной даты, когда календарь только мешает, — оставьте ввод по маске в поле;
  • для выбора из 2–6 фиксированных периодов (Неделя / Месяц / Год) — это SegmentControl.

Анатомия

Поверхность из трёх зон: шапка с навигацией, сетка дней и опциональный футер с подтверждением.

  • Заголовок — месяц и год; клик открывает выбор года → месяца
  • Навигация — стрелки «назад / вперёд» листают месяцы (.ibtn)
  • Дни недели — строка Пн…Вс, Body XS, --text-inactive
  • Сетка дней — 6×7 ячеек 40px; сегодня, выбор, диапазон, недоступно
  • Футер — опционально: «Сегодня» слева, «Отмена / Готово» справа

Варианты

Оси конфигурации: режим выбора, текущее представление и наличие футера.

Режимы выбора

Одиночная дата, диапазон дат и выбор месяца целиком.

Представления

Три внутренних представления одной поверхности: дни, месяцы, годы. Переключаются кликом по заголовку и стрелками.

С футером и без

Без футера выбор применяется сразу (docked). Футер добавляют, когда нужно подтверждение (Отмена / Готово) или быстрый переход к сегодняшней дате.

Размеры

Один размер поверхности: фиксированная ширина 304px (7 ячеек по 40px + паддинг 12px), ячейка дня 40×40, кружок числа 36px. Меняется не размер, а способ размещения: floating (всплывает над полем, с тенью) и inline (встроен в контейнер, без тени, с рамкой).

Контент

Правила контента:

  • неделя начинается с понедельника, дни недели — двухбуквенные (Пн…Вс);
  • заголовок — месяц в именительном падеже и год: «Октябрь 2025»;
  • дни соседних месяцев показываются приглушённо и остаются кликабельными (переводят на свой месяц);
  • границы min / max делают даты вне диапазона недоступными (disabled), а не скрывают их.

Здесь доступны только дни с 3 по 27 октября — остальные недоступны.

Поведение

Навигация по периодам, выбор диапазона и позиционирование относительно поля.

Навигация

Стрелки листают на месяц назад / вперёд. Клик по заголовку открывает выбор года, затем месяца (представления year → month → day). В представлении года стрелки листают на 12 лет.

Выбор диапазона

Первый клик задаёт начало (конец сбрасывается), второй — конец. Клик по дате раньше начала перезапускает выбор с новой начальной даты. Промежуток между концами подсвечивается полосой --primary-bg.

Позиционирование

В режиме floating календарь всплывает под полем с зазором и авто-разворотом вверх, если внизу не хватает места, — приём идентичен Popover (role="dialog", aria-modal="false"). Одновременно открыт только один календарь. В режиме inline поверхность встроена в контейнер и не позиционируется.

Состояния

Состояния — на уровне ячейки дня. Наведение даёт мягкую заливку кружка, сегодня — обводка --primary, выбор и концы диапазона — заливка --primary с белым числом, промежуток — полоса, недоступные даты приглушены и не реагируют.

Доступность

Поверхность — role="dialog" с aria-modal="false" и aria-label="Выбор даты". Сетка — role="grid", ячейки — role="gridcell" с полным aria-label даты («15 октября 2025»), выбранная несёт aria-selected="true". Недоступные — aria-disabled="true". Стрелки и заголовок — настоящие кнопки с aria-label.

Цвета

Только семантические токены. Поверхность — --bg-popup с тенью --elevation-5; выбор и концы диапазона — --primary; промежуток — --primary-bg; сегодня — обводка --primary.

Для разработчиков

Хендофф-пакет: точные размеры поверхности и алгоритм построения сетки/выбора. Календарь framework-agnostic — верстка на токенах, логика описана псевдокодом.
