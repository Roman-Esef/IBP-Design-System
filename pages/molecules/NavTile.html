NavTile

Навигационная плитка главной страницы платформы: иллюстрация, название раздела и краткое описание. Клик по плитке — переход на страницу раздела. Вариант со ссылками добавляет быстрые переходы к подразделам. Иллюстрации — из библиотеки Иллюстрации. Не путать с дашбордом: NavTile не показывает данные и графики — только вход в раздел.

Конструктор

Соберите плитку: вариант, наполнение и состояние.

Использование

NavTile живёт только на главной странице платформы и образует сетку разделов: минимальная ширина плитки — 400px, гэп между плитками — 16px. Плитки, не помещающиеся в ряд, переносятся на следующий (`auto-fill` / `minmax(400px, 1fr)`). Высота плиток в ряду выравнивается по самой высокой.

Сетка главной страницы: базовые плитки и плитка со ссылками, 2×2.

Когда использовать

  • вход в основной раздел платформы с главной страницы — одна плитка = один раздел;
  • раздел с частыми подразделами — вариант со ссылками (быстрые переходы без промежуточной страницы).

Когда не использовать

  • показ показателей, метрик и графиков — это дашборд (отдельная страница платформы), а не NavTile;
  • навигация внутри раздела — используйте Tab или NavPanel;
  • карточка сущности со статусом и действиями — Tile.

Соседние компоненты: Tile — карточка объекта данных в списках; NavPanel — постоянная боковая навигация; NavTile — разовая точка входа с главной.

Анатомия

Раскладка горизонтальная: слева вертикальный стек — название, описание, в варианте со ссылками — блок ссылок, прижатый к низу; справа — иллюстрация фиксированного размера 195×140, прижатая к правому краю плитки и центрированная по вертикали. Обязательно только название.

  • Контейнер — фон --bg-tile, рамка 1px --border-light, радиус --radius-control (8px), паддинг 24px, по умолчанию 483×253px
  • Иллюстрация — слот .illu, фиксированный размер 195×140, вплотную к правому краю и с отступом сверху (24px), из библиотеки Иллюстрации; опциональна
  • Название — H4 Strong; в варианте со ссылками обёрнуто в ссылку .ntile__title-link
  • Описание — Body M, --text-secondary; 1–2 строки, опционально
  • Ссылки — до 4 ссылок .link--accent .link--m, только в варианте со ссылками

Варианты

Одна ось конфигурации — способ навигации.

Базовая

Вся плитка — одна ссылка <a class="ntile"> на страницу раздела. Внутри интерактивных элементов нет.

Со ссылками

Контейнер — <div class="ntile ntile--links">, НЕ ссылка (вложение <a> в <a> недопустимо). На страницу раздела ведёт название-ссылка; ниже — до 4 ссылок на подразделы, прижаты к низу плитки. Hover-подъём у контейнера отсутствует — реагируют только ссылки.

Размеры

Один размер — 442×188px по умолчанию (вне сетки), минимальная ширина 400px (не сжимается меньше). Внутри .ntile-grid ширина растягивается по колонке (`auto-fill, minmax(400px, 1fr)`, гэп 16px) — плитки, не помещающиеся в ряд, переносятся на следующий. Минимальная высота 188px; плитки одного ряда выравниваются по самой высокой.

Контент

Правила текста:

  • название = название раздела в навигации, существительное 1–3 слова: «Сделки», не «Перейти к сделкам»;
  • описание — что внутри раздела, 1–2 строки без точки в конце; длиннее двух строк — обрезается многоточием;
  • ссылки — названия подразделов, до 4 штук; больше — оставить самые частые, остальное живёт внутри раздела;
  • иллюстрация — из библиотеки Иллюстрации, одна на раздел, не декоративная иконка.

Поведение

Навигация

Базовая плитка: клик в любую точку → переход на страницу раздела. Плитка со ссылками: клик по названию → страница раздела, клик по ссылке → подраздел; клик по фону плитки ничего не делает. Переход — обычная навигация в текущей вкладке.

Адаптивность

Сетка — `grid-template-columns: repeat(auto-fill, minmax(400px, 1fr))`, гэп 16px по умолчанию. Плитка не сжимается меньше 400px; не помещается в ряд — переносится на следующий, без брейкпоинтов и контейнерных запросов. Высота плитки растёт от контента; в ряду плитки выравниваются по самой высокой (grid stretch).

Состояния

Четыре состояния базовой плитки. У варианта со ссылками состояния контейнера отсутствуют — Hover/Focus живут на названии и ссылках. Disabled — раздел недоступен пользователю (нет прав или не подключён): плитка видна, но не кликабельна.

Доступность

Базовая плитка — обычная ссылка <a>; имя ссылки — название раздела (описание — часть контента ссылки). Иллюстрация декоративна: aria-hidden="true". Disabled — aria-disabled="true" без href, плитка исключается из Tab-обхода. Фокус — видимая обводка 2px --primary.
