SegmentControl

Компактный контрол для переключения одного значения параметра — периода, единиц, режима отображения — из 2–6 взаимоисключающих опций. Сегменты лежат в общем треке (rail); выбранный сегмент отмечен плавающим индикатором (thumb), который анимированно переезжает при переключении. Размеры M, S, XS, опциональная ведущая иконка и вариант «только иконки».

Конструктор

Соберите контрол: размер, количество сегментов, контент, состояние и Fullwidth — превью и код обновляются мгновенно. Кликните по сегменту в превью, чтобы увидеть анимацию индикатора.

Использование

SegmentControl переключает значение одного параметра в пределах текущего контекста — период графика, единицы измерения, режим отображения списка. В отличие от навигационных сущностей он не меняет набор разделов страницы и не открывает новый раздел контента — он выбирает, как показать один и тот же контент.

  • Период / диапазон. Переключатель над графиком или таблицей: День / Неделя / Месяц / Год.
  • Режим отображения. Список / Плитка / График — компактный вариант с иконками, часто в углу карточки или тулбара.
  • Взаимоисключающий выбор. Всегда выбран ровно один сегмент — как радиокнопки, но в компактной горизонтальной форме.
  • 2–6 сегментов в одном контроле. Больше — переносите в Select.

SegmentControl, Tab и ButtonGroup — в чём разница

Все три компонента внешне похожи на «ряд опций», но решают разные задачи. Выбирайте по смыслу действия, а не по внешнему виду.

SegmentControl Этот компонент

Переключает значение параметра — период, единицы, режим отображения. Общий трек с фоном и один плавающий индикатор. role="radiogroup".

Tab

Переключает раздел / вьюху страницы или модалки — меняется набор контента целиком. Индикатор — тонкая полоса на общем рейле, без фона-«трека». role="tablist".

ButtonGroup toggle

Набор независимых кнопок с общей рамкой — годится, когда сегменты это по сути Button с заливкой Accent и своей семантикой действия, без общего трека. aria-pressed.

Анатомия

Контрол — это общий трек с сегментами и одним индикатором, который абсолютно спозиционирован под текущим выбранным сегментом и анимированно едет к новому при переключении.

  • Трек (rail)полупрозрачная нейтральная заливка — остаётся читаемым на любом фоне, включая серый фон страницы
  • Индикатор (thumb)плавающая заливка выбранного сегмента — белый + мягкая тень
  • Сегменттекст и/или ведущая иконка — интерактивная зона
  • Разделительтонкая линия между соседними невыбранными сегментами — скрыта у индикатора
  • Счётчик (опционально)Badge в варианте без заливки — количество элементов внутри сегмента

Размеры

Три размера, высота и типографика согласованы с Button того же размера, чтобы контрол ровно вставал в один ряд с кнопками в тулбаре. M — базовый, над графиками и таблицами. S — плотные тулбары и карточки виджетов. XS — инлайн-контролы рядом с текстом.

Количество сегментов и равная ширина

От 2 до 6 сегментов в одном контроле — больше усложняет считывание и требует замены на Select. По умолчанию ширина каждого сегмента зависит от контента; модификатор Fullwidth растягивает контрол на всю ширину контейнера с равными сегментами — удобно для двух-трёх опций на всю ширину карточки или нижней панели модалки.

Состояния

Невыбранный сегмент проходит через Default и Hover; выбранный сегмент подсвечен индикатором и не имеет отдельного hover — он уже визуально выделен. Disabled применяется на весь контрол целиком.

Отключение одного сегмента Расширение

Частный случай: один сегмент временно недоступен (например, тариф вне подписки), при этом остальной контрол активен. Такой сегмент нельзя выбрать и сфокусировать — используйте точечно, по умолчанию Disabled действует на весь контрол.

Анимация индикатора

При переключении индикатор анимированно едет к новому сегменту и меняет ширину под его контент — transform и width, 180 мс, cubic-bezier(.4,0,.2,1). Анимация отключается при prefers-reduced-motion: reduce, оставляя только смену прозрачности.

Правила

SegmentControl переключает значение одного параметра, а не разделы страницы. Каждая пара «так / не так» раскрывает одну конкретную ситуацию.

Так

Короткие взаимоисключающие значения одного параметра — день / неделя / месяц.

Не так

Длинные названия разделов — это задача Tab, а не SegmentControl.

Так

2–6 сегментов — контрол легко считывается целиком.

Не так

7 и больше сегментов не помещаются в трек без переноса — замените на Select.

Дополнительно — без пары «не так»

Так

Fullwidth уместен для пары взаимоисключающих действий на всю ширину контейнера.

Доступность Расширение

Контейнер получает role="radiogroup" и осмысленный aria-label (например «Период»); каждый сегмент — role="radio" с aria-checked. Фокус — по одному активному сегменту (roving tabindex), переключение между сегментами и смена выбора — стрелками ← → (или ↑ ↓), Home / End — к первому/последнему. Сегмент только с иконкой обязан иметь aria-label.

Цвета

Трек — полупрозрачная нейтральная заливка (Swamp_400, 16%) — так контрол остаётся отчётливым на любой подложке, включая серый фон страницы, а не только на белой плашке. Индикатор — белая заливка BG_Tile с мягкой тенью (та же тень, что и активный чип сегмент-контрола в спецификации теней ДС). Текст невыбранного сегмента — Text_Secondary, при наведении — Text_Primary с лёгкой заливкой Active·PrimaryBG; выбранный — Primary (Emerald). Счётчик — Badge без заливки (--text), цвет наследует Text_Primary. Disabled — на рампе StDisabled.

Для разработчиков Расширение

Эта страница ДС собрана на чистых CSS-классах и vanilla JS — но всё ниже не привязано к стеку: команда, которая верстает компонент на React (или любом другом фреймворке), может взять отсюда анатомию, точные размеры, состояния и цвета, чтобы собрать свою реализацию, не заглядывая в этот код построчно.

Поведение · псевдокод (framework-agnostic) 
// 1. Позиция индикатора = позиция выбранного сегмента внутри трека.
//    Это НЕ фиксированные проценты — ширина сегмента зависит от контента
//    (текст/иконка/бейдж), поэтому позицию нужно ИЗМЕРЯТЬ, а не вычислять по индексу:
selectedEl = track.children[selectedIndex] // ref на DOM-узел выбранного сегмента
thumb.width     = selectedEl.offsetWidth
thumb.translateX = selectedEl.offsetLeft

// 2. Пересчитывать при: смене value, смене набора items/size,
//    ResizeObserver на треке (адаптивная ширина) и после загрузки шрифтов
//    (метрики шрифта меняют offsetWidth уже после первого рендера).

// React: измерение — побочный эффект после layout, не во время рендера
useLayoutEffect(() => {
  const el = itemRefs.current[selectedIndex];
  if (!el) return;
  setThumbRect({ width: el.offsetWidth, left: el.offsetLeft });
}, [value, items, size]);

useEffect(() => {
  const ro = new ResizeObserver(() => recalc());
  ro.observe(trackRef.current);
  return () => ro.disconnect();
}, []);

// 3. Клавиатура (roving tabindex) — на контейнере [role="radiogroup"]:
//    ArrowRight/ArrowDown  → следующий доступный сегмент (зациклено)
//    ArrowLeft/ArrowUp     → предыдущий доступный сегмент (зациклено)
//    Home / End            → первый / последний доступный сегмент
//    В фокус-последовательности таба участвует только выбранный сегмент —
//    у остальных tabIndex = -1.

Это не готовый сниппет для вставки — алгоритм, который нужно реализовать поверх вашего стека (в этом ДС он написан на vanilla JS, см. segment-control.page.js, функции positionThumb и wireSegControl, если нужен рабочий референс).

Рекомендуемый API компонента (React)

Форма пропсов — предложение, а не обязательный контракт; адаптируйте под конвенции вашей кодовой базы.

SegmentControl.types.ts 
type SegmentControlSize = 'm' | 's' | 'xs';

interface SegmentControlItem {
  value: string;              // уникальный ключ сегмента
  label?: string;              // опустить для icon-only
  icon?: React.ReactNode;      // 20/18/16px по размеру
  badge?: number | string;      // счётчик — рендерить как Badge variant="text"
  disabled?: boolean;           // точечное отключение сегмента
  ariaLabel?: string;           // обязателен, если label не задан
}

interface SegmentControlProps {
  items: SegmentControlItem[];
  value: string;                              // controlled-компонент
  onChange: (value: string) => void;
  size?: SegmentControlSize;                     // по умолчанию 'm'
  fullWidth?: boolean;                         // по умолчанию false
  disabled?: boolean;                          // отключает весь контрол
  'aria-label': string;                      // обязателен на радиогруппе
}
