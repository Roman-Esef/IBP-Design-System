> Автоспека для быстрого контекста. Источник истины — CSS-файл и страница компонента. При изменении компонента обновляй эту спеку.

## Назначение
Радиокнопка позволяет выбрать всего один пункт из списка взаимоисключающих вариантов. Используется только в составе группы. Два состояния выбора — Unselected / Selected — и интерактивные состояния Default / Hover / Focus / Pressed / Disabled / Error.

## Ключевые правила (из разделов страницы)
- **Использование** — Используется только в составе группы радиокнопок и не может использоваться как единственная доступная опция — в этом случае используйте Чекбокс.
- **Поведение · Описание работы** — Клик по названию или по самой радиокнопке приводит к её выбору. Повторное нажатие не снимает выбор. Когда пользователь выбирает новый элемент, предыдущий выбор автоматически отменяется.
- **Анатомия** — У иконки радиокнопки есть охранное поле: компонент иконки имеет размер 20×20 px, а сам круглый шейп внутри него — 16×16 px. В группе сверху может быть заголовок.
- **Размеры** — У радиокнопки один размер. Геометрия шейпа привязана к интерлиньяжу Body M (20px): видимый шейп — 16px в охранном поле 20px, чтобы центрироваться по первой строке названия. Значения ниже измерены на живом экземпляре.
- **Состояния** — Два состояния выбора по вертикали и интерактивные состояния по горизонтали. Состояния Hover, Focus, Pressed и Error добавлены к исходной матрице Default / Disabled.Новое
- **Варианты · Состав** — Вспомогательные части — название и хелпер — опциональны. Радиокнопка рендерится без названия (icon-only — например, в ячейке таблицы), с названием и с хелпером. Шейп всегда выровнен по первой строке текста.
- **Ошибка и валидацияНовое** — Состояние Error появляется, когда обязательная группа не имеет выбранного пункта при отправке формы. Поскольку у радиокнопки нет промежуточного состояния, ошибка всегда относится к группе целиком: шейпы и сообщение окрашиваются в --error, рядом с заголовком обязательной группы ставится звёздочка. **Группа и обязательный выбор — две независимые опции**: группа может быть необязательной, а маркер (.rb__req) — отдельная опция.
- **Контент** — Текстовый контент радиокнопки — название, опциональный хелпер и заголовок группы. Краткий свод правил (подробно — в разделах ниже):
- **Контент · Название радиокнопки** — Название пишется с заглавной буквы, без знаков пунктуации в конце, и всегда располагается справа от радиокнопки. Название каждой радиокнопки должно чётко и кратко описывать эффект от её выбора — короткие названия позволяют делать выбор быстро и уверенно.
- **Контент · Заголовок группы** — Добавляйте заголовок к группе для ясности, если взаимосвязь пунктов неочевидна, или чтобы вынести в него повторяющиеся в пунктах слова. Заголовок начинается с заглавной, начертанием Strong того же кегля, что и название. Допустимо использование H6 Strong, если группа входит в состав другой группы. Формулируйте заголовок так, чтобы было понятно: можно выбрать только один вариант.
- **Область клика и доступностьНовое** — Видимый шейп — 16px, но кликабельная область шире: она включает название и хелпер. В группе активная область вытягивается на всю высоту строки. Клавиатура работает по правилам radio group: Tab переводит фокус на группу, стрелками ↑↓ ←→ выбор перемещается между пунктами (и сразу применяется), фокус показывается обводкой --primary.
- **Типографика** — Название набирается гарнитурой SB Sans Text токеном --type-body-m, хелпер — --type-body-xs, заголовок группы — --type-h6-strong.
- **Цвета** — Компонент использует только семантические токены. Производные оттенки (hover) и state-слои вычисляются из --primary через color-mix.

### Разметка · HTML (эталонная реализация ДС)

```
<!-- группа радиокнопок: заголовок + пункты с общим name -->
<div class="rb-group" role="radiogroup" aria-labelledby="grp-t">
  <p class="rb-group__title" id="grp-t">Обновлять список</p>
  <div class="rb-group__items">
    <label class="rb rb--selected">
      <input type="radio" class="rb__input" name="refresh" checked>
      <span class="rb__box"><span class="rb__mark"></span></span>
      <span class="rb__content"><span class="rb__label">Ежеминутно</span></span>
    </label>
    <label class="rb rb--unselected">
      <input type="radio" class="rb__input" name="refresh">
      <span class="rb__box"><span class="rb__mark"></span></span>
      <span class="rb__content"><span class="rb__label">Ежечасно</span></span>
    </label>
  </div>
</div>

<!-- обязательная группа без выбора — ошибка -->
<div class="rb-group" role="radiogroup" aria-labelledby="grp-t2" aria-describedby="grp-e2">
  <p class="rb-group__title" id="grp-t2">Способ доставки<span class="rb__req">*</span></p>
  <div class="rb-group__items">…</div>
  <p class="rb-group__error" id="grp-e2"><svg aria-hidden="true">…</svg><span>Выберите один из вариантов</span></p>
</div>
```

### Поведение · псевдокод (framework-agnostic)

```
// 1. Состояние выбора — бинарное на каждом input:
//    unselected → checked=false; selected → checked=true
//    у радиокнопки нет indeterminate — всегда ровно один выбранный пункт в группе

// 2. Взаимоисключающий выбор: все input в группе делят один name —
//    браузер снимает выбор с остальных автоматически; повторный клик по выбранному не снимает выбор

// 3. Клик по названию ИЛИ шейпу переключает выбор (<label> оборачивает input)

// 4. Клавиатурная навигация (native radiogroup semantics):
//    Tab входит/выходит из группы одним шагом — фокусируется только checked-пункт (или первый, если ничего не выбрано);
//    ↑↓←→ перемещают фокус МЕЖДУ пунктами группы И сразу применяет выбор (без Enter/Space)

// 5. Интерактивные состояния — чистый CSS (:hover / :active / :focus-visible);
//    hover: рамка primary + halo 4px; pressed: halo 6px
//    классы .rb--hover / .rb--focus / .rb--pressed — только для документации

// 6. Error: обязательная группа без выбора при сабмите → класс .rb--error на всех пунктах группы;
//    сообщение .rb-group__error (Body XS, --error), связано через aria-describedby;
//    выбор любого пункта снимает ошибку со всей группы

// 7. Disabled: input.disabled + .rb--disabled (pointer-events: none) — может быть у отдельных
//    пунктов внутри доступной группы
```

### Radiobutton.types.ts

```
type RadioOrientation = 'vertical' | 'horizontal';

interface RadioProps extends Omit<React.InputHTMLAttributes<HTMLInputElement>, 'type' | 'onChange'> {
  value: string;                // значение пункта в группе
  label?: React.ReactNode;        // название справа от шейпа
  helper?: React.ReactNode;       // подпись Body XS под названием
  disabled?: boolean;
}

interface RadioGroupProps {
  name: string;                 // общий name для всех радиокнопок группы
  value: string | null;      // value выбранного RadioProps
  onChange(value: string): void;
  title?: React.ReactNode;        // заголовок группы (H6 Strong)
  orientation?: RadioOrientation; // по умолчанию 'vertical'
  required?: boolean;            // звёздочка * у заголовка
  error?: React.ReactNode;        // сообщение об ошибке уровня группы
  children: React.ReactNode;      // Radio[]
}
```

### Справочник классов и атрибутов

| Класс / атрибут | На чём | Назначение |
|---|---|---|
| .rb | label | Корень: inline-flex, зазор 8px, выравнивание по верхней строке |
| .rb--unselected / --selected | label | Состояние выбора |
| .rb--hover / --focus / --pressed | label | Форс-состояния — только для спецификации в документации, не для продакшена |
| .rb--error | label | Невалидно (от группы): красный шейп и точка (--error) |
| .rb--disabled | label | Отключено: pointer-events: none, текст/шейп в --disabled |
| .rb--no-content | label | Шейп без названия/хелпера (icon-only) |
| .rb__input | input | Визуально скрытый нативный radio — общий name даёт взаимоисключающий выбор |
| .rb__box | span | Охранное поле 20px, центрирует шейп |
| .rb__mark | span | Видимый круг 16px: рамка, внутренняя точка (::after); все state-слои на нём |
| .rb__content | span | Колонка название + хелпер (зазор 2px) |
| .rb__label | span | Название (Body M) |
| .rb__req | span | Звёздочка обязательной группы (--error) |
| .ds-helper.ds-helper--left | span | Хелпер (Body XS) — общий компонент, см. Label / Helper |
| .rb-group | div | Контейнер группы; парой — role="radiogroup" + aria-labelledby |
| .rb-group--horizontal | div | Горизонтальная раскладка пунктов (зазор 24px) |
| .rb-group__title | p | Заголовок группы (H6 Strong) |
| .rb-group__items | div | Колонка/строка пунктов |
| .rb-group--indent | div | Отступ дочерних 28px |
| .rb-group__error | p | Сообщение об ошибке уровня группы (Body XS, --error) |
| role="radiogroup" / aria-labelledby / aria-describedby | div | Доступность: связь группы с заголовком и сообщением об ошибке |
