> Автоспека для быстрого контекста. Источник истины — CSS-файл и страница компонента. При изменении компонента обновляй эту спеку.

## Назначение
Разделительная черта — тонкая линия, которая визуально отделяет одни части интерфейса от других. Используется как часть молекул (заголовки и подвалы модальных окон, списки, меню) и как самостоятельный компонент. Горизонтальный или вертикальный; цвет и толщина при смене ориентации сохраняются.

## Ключевые правила (из разделов страницы)
- **Использование** — Разделители встроены в другие компоненты — модальное окно выбора КНП: горизонтальные включены в шапку и подвал, вертикальные разделяют колонки. В списках уместен Inset-вариант, в меню — Full, в тулбаре — вертикальные линии между группами действий. Линию переиспользуют и молекулы: Context Menu (.menu__divider) и Select · DropdownList (.ddl__divider) берут цвет/толщину из токенов Divider (--dvd-color / --dvd-thickness) — единый источник.
- **Использование** — Используйте разделитель, когда нужно:
- **Варианты · Ориентация** — Разделитель может быть горизонтальным или вертикальным. Цвет и толщина линии при смене ориентации сохраняются. Горизонтальный тянется на всю ширину контейнера, вертикальный — на всю высоту строки (минимум 16px).
- **Анатомия** — Базовый разделитель — линия толщиной 1px на токене цвета. Линия рисуется бордером, поэтому без проблем переключается в пунктир. Содержит охранные отступы сверху и снизу, чтобы не прилипать к контенту.
- **Размеры** — Геометрия разделителя задаётся толщиной линии и отступами; собственных размеров у компонента нет. Значения измерены на живом экземпляре.
- **Варианты отступовНовое** — Для горизонтального разделителя есть три варианта по отступам — по аналогии с Material Design. Full делит независимые блоки на всю ширину; Inset сдвигает линию вправо, выравнивая её по тексту строки с иконкой или аватаром; Middle отступает с обеих сторон для лёгкого деления внутри блока.
- **Разделитель с подписьюНовое** — Линия может нести короткую подпись — название группы, дату в ленте или «или» между альтернативными действиями. Подпись выравнивается по центру или прижимается к краю.
- **Толщина и контрастНовое** — По умолчанию линия — 1px на светлом токене --border-light. Для более заметной границы между крупными зонами используйте контрастную линию --border-primary. Для деления длинной страницы на независимые разделы — толстый блок-секцию 8px.
- **Размеры · Отступы** — Разделитель сам по себе не задаёт вертикальный ритм — отступы до контента задаёт окружение. Держите зазор сверху и снизу симметричным, иначе линия «прилипает» к одному из блоков и связь читается неверно.
- **Контент** — Единственный текстовый контент разделителя — подпись у разделителя с текстом (см. «Разделитель с подписью» выше). Правила:
- **Использование · Рекомендации** — Не ставьте разделитель между каждой строкой — это создаёт «забор» и снижает читаемость. Группируйте связанные строки пробелом и разделяйте линией только сами группы.
- **ДоступностьНовое** — Горизонтальный разделитель — это нативный <hr>, который уже имеет роль separator. Вертикальному и текстовому разделителю на <div> присваивается role="separator", а вертикальному — ещё и aria-orientation="vertical". Чисто декоративную линию (например, фоновую границу карточки) можно скрыть от скринридера через aria-hidden="true", чтобы не засорять навигацию.
- **Цвета · Токены** — Компонент настраивается через CSS-переменные. Все значения по умолчанию ссылаются на семантические токены дизайн-системы.
- **Типографика** — Подпись разделителя набирается гарнитурой SB Sans Text токеном --type-body-s.
- **Цвета** — Компонент использует только семантические токены границ и текста. Никаких собственных цветов.

## Для разработчиков (выжимка)

### Точные размеры (redline)

Таблица рендерится на странице через getComputedStyle. Точные значения — в CSS-файле компонента (см. `css:` в шапке).

### Разметка · HTML (эталонная реализация ДС)

```
<!-- горизонтальный: нативный <hr>, уже имеет role=separator -->
<hr class="dvd dvd--h">

<!-- варианты отступов (только горизонтальный) -->
<hr class="dvd dvd--h dvd--inset">   <!-- Inset -->
<hr class="dvd dvd--h dvd--middle">  <!-- Middle -->
<hr class="dvd dvd--h dvd--section"> <!-- блок-секция 8px -->

<!-- вертикальный: <div>, role/aria вручную -->
<div class="dvd dvd--v" role="separator" aria-orientation="vertical"></div>

<!-- с подписью (только горизонтальный) -->
<div class="dvd-text dvd-text--center" role="separator">
  <span class="dvd-text__label">или</span>
</div>

<!-- чисто декоративная линия (напр. граница карточки) — скрыть от скрин-ридера -->
<hr class="dvd dvd--h" aria-hidden="true">
```

### Поведение · псевдокод (framework-agnostic)

```
// 1. Компонент полностью декларативен — нет состояний, фокуса или взаимодействия

// 2. Горизонтальный <hr> имеет встроенную роль separator — дополнительный role не нужен;
//    вертикальный на <div> требует явный role="separator" + aria-orientation="vertical"

// 3. Выбор варианта отступа (только горизонтальный): Full — независимые блоки,
//    Inset — если слева есть иконка/аватар и линию надо выровнять по тексту,
//    Middle — лёгкое деление внутри одного блока

// 4. С подписью: два сегмента линии (::before / ::after) гибко растягиваются по краям;
//    выравнивание left/right/center задаёт flex-basis короткого сегмента = --dvd-inset

// 5. Декоративные разделители (например, граница карточки вместо <hr>) —
//    aria-hidden="true", чтобы не засорять навигацию скрин-ридера
```

### Divider.types.ts

```
type DividerOrientation = 'horizontal' | 'vertical';
type DividerVariant = 'full' | 'inset' | 'middle' | 'section';

interface DividerProps {
  orientation?: DividerOrientation; // по умолчанию 'horizontal'
  variant?: DividerVariant;         // по умолчанию 'full'; игнорируется для vertical
  strong?: boolean;               // контрастная линия (--border-primary)
  dashed?: boolean;
  label?: React.ReactNode;          // подпись; только для horizontal, не section
  align?: 'left' | 'center' | 'right'; // выравнивание подписи
  decorative?: boolean;            // true — aria-hidden вместо role="separator"
}
```

### Справочник классов и атрибутов

| Класс / атрибут | На чём | Назначение |
|---|---|---|
| .dvd | hr / div | Базовый класс: линия рисуется бордером |
| .dvd--h / --v | hr / div | Ориентация |
| .dvd--inset / --middle | hr | Отступы (только горизонтальный) |
| .dvd--section | hr / div | Толстый блок-секция 8px, заливка --tertiary |
| .dvd--strong | hr / div | Контрастный цвет линии (--border-primary) |
| .dvd--dashed | hr / div | Пунктирный стиль границы |
| .dvd-text / --left / --center / --right | div | Разделитель с подписью и выравниванием |
| .dvd-text__label | span | Текст подписи (Body S) |
| role="separator" / aria-orientation | div | Обязательно для вертикального/текстового разделителя на <div> |
| aria-hidden="true" | hr / div | Скрывает чисто декоративную линию от скрин-ридера |
