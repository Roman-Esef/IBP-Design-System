> Автоспека для быстрого контекста. Источник истины — CSS-файл и страница компонента. При изменении компонента обновляй эту спеку.

## Назначение
Компактный контрол для переключения одного значения параметра — периода, единиц, режима отображения — из 2–6 взаимоисключающих опций. Сегменты лежат в общем треке (rail); выбранный сегмент отмечен плавающим индикатором (thumb), который анимированно переезжает при переключении. Размеры M, S, XS, опциональная ведущая иконка и вариант «только иконки».

## Ключевые правила (из разделов страницы)
- **Использование** — SegmentControl переключает значение одного параметра в пределах текущего контекста — период графика, единицы измерения, режим отображения списка. В отличие от навигационных сущностей он не меняет набор разделов страницы и не открывает новый раздел контента — он выбирает, как показать один и тот же контент.
- **Использование · SegmentControl, Tab и ButtonGroup — в чём разница** — Все три компонента внешне похожи на «ряд опций», но решают разные задачи. Выбирайте по смыслу действия, а не по внешнему виду.
- **Анатомия** — Контрол — это общий трек с сегментами и одним индикатором, который абсолютно спозиционирован под текущим выбранным сегментом и анимированно едет к новому при переключении.
- **Размеры** — Три размера, высота и типографика согласованы с Button того же размера, чтобы контрол ровно вставал в один ряд с кнопками в тулбаре. M — базовый, над графиками и таблицами. S — плотные тулбары и карточки виджетов. XS — инлайн-контролы рядом с текстом.
- **Контент** — Сегмент — это текст, иконка с текстом, только иконка или та же связка с числовым счётчиком. Для сегмента только с иконкой обязателен aria-label — видимого текста нет.
- **Поведение · Количество сегментов и равная ширина** — От 2 до 6 сегментов в одном контроле — больше усложняет считывание и требует замены на Select. По умолчанию ширина каждого сегмента зависит от контента; модификатор Fullwidth растягивает контрол на всю ширину контейнера с равными сегментами — удобно для двух-трёх опций на всю ширину карточки или нижней панели модалки.
- **Состояния** — Невыбранный сегмент проходит через Default и Hover; выбранный сегмент подсвечен индикатором и не имеет отдельного hover — он уже визуально выделен. Disabled применяется на весь контрол целиком.
- **Поведение · Отключение одного сегмента** — Частный случай: один сегмент временно недоступен (например, тариф вне подписки), при этом остальной контрол активен. Такой сегмент нельзя выбрать и сфокусировать — используйте точечно, по умолчанию Disabled действует на весь контрол.
- **Поведение · Анимация индикатора** — При переключении индикатор анимированно едет к новому сегменту и меняет ширину под его контент — transform и width, 180 мс, cubic-bezier(.4,0,.2,1). Анимация отключается при prefers-reduced-motion: reduce, оставляя только смену прозрачности.
- **Типографика** — Текст сегмента набирается токенами кнопок — SB Sans Screen, начертание Regular, во всех состояниях включая выбранный.
- **Использование · Правила** — SegmentControl переключает значение одного параметра, а не разделы страницы. Каждая пара «так / не так» раскрывает одну конкретную ситуацию.
- **Доступность** — Контейнер получает role="radiogroup" и осмысленный aria-label (например «Период»); каждый сегмент — role="radio" с aria-checked. Фокус — по одному активному сегменту (roving tabindex), переключение между сегментами и смена выбора — стрелками ← → (или ↑ ↓), Home / End — к первому/последнему. Сегмент только с иконкой обязан иметь aria-label.
- **Цвета** — Трек — полупрозрачная нейтральная заливка (Swamp_400, 16%) — так контрол остаётся отчётливым на любой подложке, включая серый фон страницы, а не только на белой плашке. Индикатор — белая заливка BG_Tile с мягкой тенью (та же тень, что и активный чип сегмент-контрола в спецификации теней ДС). Текст невыбранного сегмента — Text_Secondary, при наведении — Text_Primary с лёгкой заливкой Active·PrimaryBG; выбранный — Primary (Emerald). Счётчик — Badge без заливки (--text), цвет наследует Text_Primary. Disabled — на рампе StDisabled.

## Для разработчиков (выжимка)

### Точные размеры (redline)

Таблица рендерится на странице через getComputedStyle. Точные значения — в CSS-файле компонента (см. `css:` в шапке).

### Разметка · HTML (эталонная реализация ДС)

```
<!-- thumb идёт первым ребёнком — сегменты .segctrl__item начинаются со второго -->
<div class="segctrl segctrl--m" role="radiogroup" aria-label="Период">
  <div class="segctrl__thumb"></div>

  <button type="button" class="segctrl__item" role="radio" aria-checked="false" tabindex="-1">
    <span class="segctrl__label">Неделя</span>
  </button>

  <button type="button" class="segctrl__item" role="radio" aria-checked="true" tabindex="0">
    <span class="segctrl__label">Месяц</span>
    <!-- опциональный счётчик — реальный Badge, вариант --text (без заливки) -->
    <span class="badge badge--text badge--s">12</span>
  </button>

  <button type="button" class="segctrl__item" role="radio" aria-checked="false" tabindex="-1">
    <span class="segctrl__label">Год</span>
  </button>
</div>
```

### Поведение · псевдокод (framework-agnostic)

```
// 1. Позиция индикатора = позиция выбранного сегмента внутри трека.
//    Это НЕ фиксированные проценты — ширина сегмента зависит от контента
//    (текст/иконка/бейдж), поэтому позицию нужно ИЗМЕРЯТЬ, а не вычислять по индексу:
selectedEl = track.children[selectedIndex] // ref на DOM-узел выбранного сегмента
thumb.width     = selectedEl.offsetWidth
thumb.translateX = selectedEl.offsetLeft

// 2. Пересчитывать при: смене value, смене набора items/size,
//    ResizeObserver на треке (адаптивная ширина) и после загрузки шрифтов
//    (метрики шрифта меняют offsetWidth уже после первого рендера).

// React: измерение — побочный эффект после layout, не во время рендера
useLayoutEffect(() => {
  const el = itemRefs.current[selectedIndex];
  if (!el) return;
  setThumbRect({ width: el.offsetWidth, left: el.offsetLeft });
}, [value, items, size]);

useEffect(() => {
  const ro = new ResizeObserver(() => recalc());
  ro.observe(trackRef.current);
  return () => ro.disconnect();
}, []);

// 3. Клавиатура (roving tabindex) — на контейнере [role="radiogroup"]:
//    ArrowRight/ArrowDown  → следующий доступный сегмент (зациклено)
//    ArrowLeft/ArrowUp     → предыдущий доступный сегмент (зациклено)
//    Home / End            → первый / последний доступный сегмент
//    В фокус-последовательности таба участвует только выбранный сегмент —
//    у остальных tabIndex = -1.
```

### SegmentControl.types.ts

```
type SegmentControlSize = 'm' | 's' | 'xs';

interface SegmentControlItem {
  value: string;              // уникальный ключ сегмента
  label?: string;              // опустить для icon-only
  icon?: React.ReactNode;      // 20/18/16px по размеру
  badge?: number | string;      // счётчик — рендерить как Badge variant="text"
  disabled?: boolean;           // точечное отключение сегмента
  ariaLabel?: string;           // обязателен, если label не задан
}

interface SegmentControlProps {
  items: SegmentControlItem[];
  value: string;                              // controlled-компонент
  onChange: (value: string) => void;
  size?: SegmentControlSize;                     // по умолчанию 'm'
  fullWidth?: boolean;                         // по умолчанию false
  disabled?: boolean;                          // отключает весь контрол
  'aria-label': string;                      // обязателен на радиогруппе
}
```

### Справочник классов и атрибутов

| Класс / атрибут | На чём | Назначение |
|---|---|---|
| .segctrl | контейнер | Трек — role="radiogroup", обязателен aria-label |
| .segctrl--m / --s / --xs | контейнер | Размер: высота, паддинги, типографика, иконка |
| .segctrl--fullwidth | контейнер | Сегменты растягиваются на равную ширину на 100% контейнера |
| .segctrl--disabled / [aria-disabled="true"] | контейнер | Отключает весь контрол целиком |
| .segctrl__thumb | div, первый ребёнок | Плавающий индикатор; ширина/позиция задаются JS (positionThumb) |
| .segctrl__item | button | Сегмент — role="radio", roving tabindex |
| .segctrl__item--icon-only | button | Сегмент без текста — обязателен aria-label на кнопке |
| .segctrl__label | span | Текст сегмента, обрезается многоточием при нехватке места |
| [aria-checked="true"] | button | Выбранный сегмент — под ним стоит индикатор |
| [aria-disabled="true"] | button (точечно) | Отключает один сегмент, остальной контрол активен |
| .badge.badge--text.badge--{s\|xs\|xxs} | span в сегменте | Опциональный счётчик содержимого — компонент Badge, вариант без заливки |
