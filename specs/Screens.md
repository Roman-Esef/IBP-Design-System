# Примеры экранов

Живые мокапы реальных экранов Investment Banking Platform как эталон применения ДС. **Статичные** (не кликабельные прототипы). Данные условны; часть UI может устаревать.

## Файлы
| № | Файл | Экран |
|---|---|---|
| 01 | pages/screens/01-Portfolio.html | Текущий портфель ДИД — реестр сделок (широкая таблица статусов, закреплённые колонки, пагинация) |
| 02 | pages/screens/02-Deal.html | Карточка сделки — КНР, сроки, команда, метрики, дерево продуктов, ФИ |
| 03 | pages/screens/03-DealFixModal.html | Карточка сделки + модалка «Фиксация ЭПС/ВБС» |

| 05 | pages/screens/05-Instrument.html | ФИ: Доп. доходность — карточка финансового инструмента |

| 09 | pages/screens/09-CorpRequest.html | Корпоративный запрос — карточка + правая AI-панель |

## Общие ассеты
- `styles/screens.css` — только каркас и раскладки, которых пока нет отдельными компонентами: шелл `.scr`/`.ws`, статус-пилюли `.st`, простая таблица `.dt` (закреплённые колонки), заглушка `.stub`, allocation-бар `.alloc`/`.leg`, вертикальный степпер `.stp`, дерево продуктов `.ptree`/`.pt-row`, карточка ФИ `.fi`, боксы ставок `.rate-box`, агрегатор CashFlow `.cf-*`, AI-панель `.aip`/`.ai-card`. Значения — только токены ДС.
- `scripts/screens-chrome.js` — вставляет свёрнутый левый рельс (`.nav--rail`) в `[data-screen-rail]`. Атрибуты: `data-active` (имя иконки активного пункта), `data-avatar` (инициалы). Требует `icons-data.js` + `ds-icons.js`.

## Заглушки-плейсхолдеры (компонентов ещё нет в ДС)
- График CashFlow (bar+line combo) — `07`.
- Allocation-бар финметрик отрисован упрощённо через `.alloc` — `02`.
- Вертикальный степпер «Заведение сделки» — `.stp`, кандидат в компоненты.
- AI-панель корпзапроса — `.aip`, кандидат в организмы.
- Полноценного компонента Table пока нет — таблицы собраны локальным `.dt`.

## Регистрация нового экрана
1. Копия ближайшего экрана в `pages/screens/`, первой строкой — карточка `@dsCard group="Примеры экранов"`.
2. Иконки — только `<i data-icon>`; рельс — `[data-screen-rail]` + `screens-chrome.js`.
3. Карточка на `index.html` (массив `S` в `#screensGrid`) + пункт в подгруппе «Примеры экранов» в `scripts/ds-nav.js`.
4. Обнови таблицу файлов в этой спеке.
